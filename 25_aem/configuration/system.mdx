### Deliverables

<AccordionGroup>
  <Accordion icon="github" title="Logo File">
    The name of the customer company logo image file in the data folder. This image will appear on the final reports.
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Area Name">
    The name of the survey area which we are performing the inversion on. This area name will appear in the final reports. 
  </Accordion>
  <Accordion icon="rectangle-terminal" title="Project ID">
    The ID of the project which we are performing the inversion on. This area name will appear in the final reports. 
  </Accordion>
</AccordionGroup>

### Transmitter

<AccordionGroup>
  <Accordion icon="github" title="Radius (tx_radius_m)">
    The radius of the transmitter loop in meters. This is used for the modelling of the signal. 
  </Accordion>
  <Accordion icon="github" title="Turns (tx_turns)">
    The number of turns in the transmitter coil. This must be an integer and it is used for the modelling of the signal. 
  </Accordion>
</AccordionGroup>

### Data

<AccordionGroup>
  <Accordion icon="github" title="Easting Column (xcol)">
    Please provide here the column name within the data file provided which indicates the Easting (in meters) of the sounding. 
  </Accordion>
  <Accordion icon="github" title="Northing Column (ycol)">
    Please provide here the column name within the data file provided which indicates the Northing (in meters) of the sounding. 
  </Accordion>
  <Accordion icon="github" title="A Column (zcol)">
    Please provide here the column name within the data file provided which indicates the EM transmitter-receiver loop terrain clearance. 
  </Accordion>
  <Accordion icon="github" title="Data Column (sfz)">
    Please provide here the prefix used within the data file provided which which provides the derivative of the magnetic field in the z direction. 
  </Accordion>
  <Accordion icon="github" title="Line Column (line_col)">
    Please provide here the column name within the data file provided which indicates the line of the sounding. 
  </Accordion>
  <Accordion icon="github" title="Topography Column (topo_col)">
    Please provide here the column name within the data file provided which indicates the digital elevation model of the sounding. 
  </Accordion>
  <Accordion icon="github" title="Epsilon (data_eps)">
    The data uncertaintiy used in the loss function. Please see the loss function details for more details. 
  </Accordion>  
  <Accordion icon="github" title="Standard Deviation (data_std)">
    The percentage data uncertainty used in the loss function. Please see the loss function details for more details. 
  </Accordion>
</AccordionGroup>
